# ZEONE Whitepaper

## Abstract
Современный интернет завязан на централизованные облака для вычислений, доставки контента и AI-инференса. Это создает узкие места, риски цензуры и высокую стоимость трафика. ZEONE предлагает децентрализованную операционную систему, где вычисления происходят рядом с данными (Compute-over-Data), а трафик усиливается через P2P-амплификацию. Узлы объединяют транспорт, экономику и когнитивные сервисы, формируя суверенный слой сетевой интеллигенции.

## Solution Overview
- **Compute-over-Data:** LLM и vision-инференс выполняются на периферии, данные не покидают узел без шифрования.
- **Traffic Amplification:** Протокол `CACHE_REQUEST`/`CACHE_RESPONSE` и Amplifier позволяют повторно использовать скачанные сегменты (видео, файлы) между пирами, снижая нагрузку на выходные узлы.
- **Decentralized VPN:** `VpnExitAgent` выполняет реальные сетевые запросы, а Pathfinder выбирает лучший выход по цене/скорости.
- **Hybrid Settlement:** Off-chain IOU для мгновенных микроплатежей + on-chain ERC-20 для периодических расчетов.

## Tokenomics
- **Token:** ZEO.
- **Emission & Burn:** Оплата услуг (VPN, CDN, AI) сжигает часть комиссии, создавая дефляционный эффект.
- **Staking для Exit Nodes:** Провайдеры VPN и CDN блокируют ZEO для повышения лимитов и приоритета в маршрутизации; штрафы за мошенничество (неверные чанки) уменьшают стейк.

## Governance
- DAO управляет параметрами сети: размеры комиссий, лимиты IOU, политики репликации и листинг доверенных моделей.
- Голосование взвешено по доле стейка и репутации (Trust Score из Ledger).
