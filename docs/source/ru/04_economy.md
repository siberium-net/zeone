# Экономика и токеномика (RU)

- **Ledger:** Балансы, IOU, Trust Score; контроль долга и блокировка злоупотреблений.
- **Микроплатежи:** Off-chain записи о долге/кредите за трафик и услуги; сверка баланса при handshake.
- **Settlement:** Периодические выплаты в ERC-20 за накопленные кредиты; IOU поддерживает бартер и кредитные линии.
- **Стейкинг для exit:** Провайдеры VPN/CDN стейкают ZEO для повышения лимитов/приоритета; нарушение уменьшает доверие и может штрафоваться.
- **Ценовые сигналы:** `VpnExitAgent` публикует `price_per_mb`; Pathfinder может выбирать самый дешевый или самый быстрый выход.
