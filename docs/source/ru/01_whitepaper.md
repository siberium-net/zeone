# ZEONE Whitepaper (RU)

## Аннотация
Централизованные облака контролируют AI-инференс и доставку трафика, создавая узкие места, риски цензуры и высокую стоимость. ZEONE — децентрализованная ОС, переносящая вычисления к данным (Compute-over-Data) и усиливающая трафик через P2P-обмен чанками. Узлы совмещают транспорт, экономику и когнитивные сервисы, формируя суверенный сетевой слой.

## Обзор решения
- **Compute-over-Data:** LLM и vision-инференс на периферии; данные не уходят в открытом виде.
- **Traffic Amplification:** Протокол `CACHE_REQUEST` / `CACHE_RESPONSE` позволяет пирам переиспользовать сегменты (видео/файлы) и разгружать exit-узлы.
- **Децентрализованный VPN:** `VpnExitAgent` исполняет реальные сетевые запросы; Pathfinder выбирает выход по скорости/цене/надежности.
- **Гибридные расчеты:** Off-chain IOU для мгновенных микроплатежей + периодические выплаты в ERC-20.

## Токеномика
- **Токен:** ZEO.
- **Сжигание:** Комиссии за VPN/CDN/AI частично сжигаются (дефляция).
- **Стейкинг для Exit:** Провайдеры VPN/CDN стейкают ZEO для лимитов и приоритета; мошенничество (неверные чанки) снижает стейк и доверие.

## Governance
- DAO настраивает параметры сети: комиссии, лимиты IOU, политики репликации, список доверенных моделей.
- Вес голосов = стейк + Trust Score (из Ledger).
